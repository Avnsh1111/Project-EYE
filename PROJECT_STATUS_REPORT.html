2Critical Bugs — Fix Before Production

BUG-1 Wrong Config Key in ProcessImageAnalysis
app/Jobs/ProcessImageAnalysis.php · Line 441

Uses config('services.python_ai.url') but config/services.php does not define this key. Archive and code file processing will silently fall back to an incorrect default URL and fail.

Fix: Change to config('ai.api_url')
BUG-2 Unsafe Range Request Regex — Potential Undefined Index
app/Http/Controllers/MediaController.php · Line 47

The regex preg_match('/bytes=(\d+)-(\d*)/', $range, $matches) does not validate if the match succeeded before accessing $matches[1]. A malformed Range header during video/audio streaming will cause an undefined index error.

Fix: Add if (empty($matches)) abort(416, 'Range Not Satisfiable'); after the preg_match call
